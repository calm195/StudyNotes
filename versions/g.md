# go版本管理

1. windows

   在[g官网](<https://github.com/voidint/g>)上下载release，解压在本地，然后添加环境变量。~~ps: 需要删除存在的go环境变量~~

   `G_HOME`:`*/g/` 是g.exe所在的目录

   `GOROOT`: `%G_HOME%/go/`

   用户变量可以新增`G_MIRROR`:`https://golang.google.cn/dl` 以便更快下载go

   `g install xx`

   `g use xx`x

   `go version`
